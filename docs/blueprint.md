# **App Name**: CampusConnect Landing Page

## Core Features:

- Hero Section: Display a compelling hero section with school name, headline, CTA, and background image.
- About Section: Showcase school mission, history, and unique values.
- Key Features Highlight: Present the key website features such as online admission, events, student portal, and contact information.
- Academic Programs: Display academic programs using simple cards for nursery, primary, secondary, and specialized programs.
- Contact Form Submission: Capture user input from the contact form and send submissions to Resend or Formspree.
- Dynamic Testimonial Generation: Use AI to generate compelling testimonials.
- Footer Section: Include copyright information, social links, address, and contact details in the footer.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to convey trust and modernity.
- Background color: Light blue (#E0F7FA) to create a clean and professional look.
- Accent color: Soft teal (#79E0E9) for interactive elements and highlights.
- Body and headline font: 'Inter' for a modern, neutral, and readable design.
- Use clear and professional icons to represent features and sections.
- Maintain a clean and well-organized layout to ensure easy navigation and readability.
- Incorporate subtle animations for smooth transitions and engaging interactions.
